# 🌪️ Disaster Tweet Classification Engine

An end-to-end Machine Learning web application designed to classify whether a given tweet is about a real disaster or not. This project uses Natural Language Processing (NLP) to analyze text and predict emergency situations in real-time.

---

## 👤 Project Details
* **Presenter:** Abhishek Sharma
* **Organization:** NEXHIKES
* **Dataset Size:** 7,613 logs (Tweets)

---

## 🛠️ Tech Stack & Files
* **Frontend:** Streamlit (Python)
* **Model:** Scikit-Learn (Logistic Regression / Naive Bayes / Random Forest)
* **Text Processing:** TF-IDF Vectorizer
* **Files Included:**
  * `app.py` - Streamlit Web Application
  * `best_disaster_model.pkl` - Trained Classifier Model
  * `tfidf_vectorizer.pkl` - Fitted TF-IDF Vectorizer
  * `requirements.txt` - Python Dependencies
